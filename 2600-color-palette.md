# Atari 2600 – Practical “Authentic” Color Palette  

This palette is a curated, designer-friendly abstraction of the Atari 2600 color look.
It is intended for modern design, UI mockups, posters, and retro-inspired games,
not for cycle-accurate hardware emulation.

There was no official fixed color scheme on the Atari 2600.
The recognizable visual identity emerged from hardware limits, CRT behavior,
and developer convention.

---

## Design Intent

- Faithful to NTSC CRT-era appearance
- Emphasizes contrast, flat color fields, and low color counts
- Uses modern HEX values that reproduce the “2600 feel”

---

---

## Authentic Usage Rules

- Target 3–5 colors per screen
- Prefer high contrast over variety
- Use flat fills only (no gradients, transparency, or soft shadows)
- Embrace unconventional combinations (orange on blue, purple skies)
